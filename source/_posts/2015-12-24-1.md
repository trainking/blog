![截图](http://7xpgwr.com1.z0.glb.clouddn.com/jitu20151224165700.png)

## 步骤一:

`pip install tqdm`

使用`pip`快速获取。

## 步骤二:

只要一小段代码就可以体验到效果：

```
    import time
    from tqdm import *
    for i in tqdm(range(1000)):
        time.sleep(.01)

```
